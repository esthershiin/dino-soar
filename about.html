How it works

We used the ml5.js library to get the body coordinates to control the game script. PoseNet, a Machine Learning model, provides real-time coordinates of the detected body. We used this data to tweak our game controls. The white dot represents the nose position, the top dotted line represents the jump line, and the bottom dotted line represents the duck line. If the nose dot is above the jump line, the character will jump and if the nose dot is below the duck line, the character will duck.
